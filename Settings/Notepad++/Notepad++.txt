32-bit (wegen Kompatibilität mit einigen Plugins, z.B. HEX-Editor, MultiClipboard, TextFX)

Einstellungen
    Optionen > Sprache
        Tabulatoren
            Tabulatorbreite: 4
            x Durch Leerzeichen ersetzen
    Stile
        Design: Solarized
        Schriftart: Cascadia Code (CaskaydiaCove Nerd Font zeigt Umlaute nicht korrekt an)
        x Schriftart als Standard setzen

https://github.com/bruderstein/nppPluginManager/releases
    x32.zip herunterladen
    Ins Notepad++ Installations-Verzeichnis entpacken

Erweiterungen > Plugin Manager > Show Plugin Manager
    - Compare
    - HEX-Editor
    - JSTool
    - MultiClipboard
    - TextFX Characters
    - XML Tools

    Bereits installiert:
    - Converter
    - DSpellCheck
    - MIME Tools
    - NppExport
    - NppFTP
    - Plugin Manager

    Außerdem interessant (ungetestet)
    - JSON Viewer
    - HTML Tag
    - Bookmark Manager
    - Poor Man's T-Sql Formatter
    - ToolBucket
    - XPatherizerNPP

Erweiterungen (manuell)
    https://sourceforge.net/projects/nppjsonviewer/
    Ins Plugin-Verzeichnis kopieren und neustarten

Themes:
    Verzeichnis themes nach %appdata%\notepad++ kopieren

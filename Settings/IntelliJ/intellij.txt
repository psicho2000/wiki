Installation
    Install Toolbox App
    Via Toolbox App, install IntelliJ IDEA
    Toolbox > Cog
        x Update Toolbox automatically
        Global shortcut to open Toolbox: Win+Ctrl+J
    Toolbox > IntelliJ > Cog > Settings > Edit Virtual Machine Options > Notepad++
        Add "-Dproject.tree.structure.show.url=false" at the end in a new line and save the file

    "defaultInstallRoot": "C:/Program Files"
}

Batch Installation:
$ cat winget-input-file | xargs --max-args=1 winget.exe install

Batch upgrades:
$ winget upgrade --all

Update deprecated Choco and add winget aliases:
`cfg edit`
```bash
alias clist=`choco.exe list`
alias cup=`choco.exe upgrade`
alias cinst=`choco.exe install`
alias cuninst=`choco.exe uninstall`
alias wg='winget.exe'
```

Ukelele
'US International - Custom.bundle' hat folgende Konfiguration:
    * basiert auf 'US International - PC'
    * 1. Ebene
        * Paragraph (links von 1)           -> `
        * Dead ` (rechts von Left Shift)    -> Undead `
        * Dead ' (rechts von ;)             -> Undead '
    * 2. Ebene (Shift)
        * +- (links von 1)                  -> ~
        * kleines ^ (6)                     -> normales ^
        * Dead ~ (rechts von Left Shift)    -> Undead ~
        * Dead " (rechts von :)             -> Undead "
    * 3. Ebene (Option)
        * 1                                 -> ¹
        * 2                                 -> ²
        * 3                                 -> ³
        * 4                                 -> ⁴
        * Num1                              -> ä
        * Num2                              -> ö
        * Num3                              -> ü
        * Num4                              -> Ä
        * Num5                              -> Ö
        * Num6                              -> ü
        * Num7                              -> ß
        * Num8                              -> €

* Always install for Current User (Installing for All Users crashes)
* Update procedure:
    1. Ukelele > File > Install > Show Organiser
        a. Set Folder... > Documents
        b. Remove from current user
    2. Copy 'US International - Custom.bundle' to Documents
    3. Ukelele > File > Install > Show Organiser
        a. Install for Current User
    4. Relogin
    5. Systemeinstellungen > Keyboard > Texteingabe > Eingabequellen > Bearbeiten...
        + > Englisch > US International - Custom > Hinzufügen

Menüleiste konfigurieren
* Hidden Bar installieren und konfigurieren
* https://www.heise.de/ratgeber/Tipp-Mehr-Platz-fuer-Menueleistensymbole-in-macOS-9850600.html
  ```
  defaults -currentHost write -g NSStatusItemSpacing -int 6
  defaults -currentHost write -g NSStatusItemSelectionPadding -int 6
  killall ControlCenter
  ```
  Then relogin
* To reset:
  ```
  defaults -currentHost delete -g NSStatusItemSpacing
  defaults -currentHost delete -g NSStatusItemSelectionPadding
  killall ControlCenter
  ```
  Then relogin
